## [1.0.0] - 2026-07-16

- Add OpenAI-compatible commit message generation from staged diffs.
- Add a commit message generation entry to the Source Control title bar.
- Add a configurable custom prompt for commit-message generation.
- Support multiline Markdown custom prompts in the Settings UI.
- Include repository Commitlint configuration in commit-message generation instructions.
- Prioritize the style of the 10 most recent repository commits.
- Use existing Git input as a generation hint without overwriting edits made during generation.
- Add a Marketplace icon for the extension.
